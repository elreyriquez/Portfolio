# Graphic Design Assets

## Folder structure

- **downloaded/**: Images pulled from Wix (run `scripts/download-graphic-design-images.sh` first)
- **Illustrations/**: Your illustration work
- **Logos/**: Logo designs (e.g. JustShipIt)
- **Events/**: Event-related graphics
- **Food/**: Food/beverage design work
- **Corporate/**: Corporate/brand work

## Workflow

1. Run the download script from project root: `./scripts/download-graphic-design-images.sh`
2. Review images in `downloaded/`
3. Move the ones you want to keep into the appropriate category folder
4. Delete any from `downloaded/` you don't need
